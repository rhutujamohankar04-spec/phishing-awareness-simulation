# Phishing Awareness Simulation Using Social Engineering Techniques

## Overview
This project demonstrates how phishing attacks work and analyzes user awareness through a simulated phishing email and login page.

The project was created as part of a cybersecurity internship to study how users respond to phishing attempts.

---

## Objectives

- Understand phishing techniques
- Simulate phishing email attacks
- Measure user awareness
- Analyze user behavior in phishing scenarios
- Recommend preventive cybersecurity practices

---

## Implementation

The project includes:

- A simulated phishing email template
- A demonstration login page created using HTML
- A Google Form survey to measure phishing awareness
- Data analysis of user responses

---

## Results

The survey results show that a significant percentage of users are unable to identify phishing emails, highlighting the need for improved cybersecurity awareness.

---

## Preventive Measures

- Multi-factor authentication
- Email filtering
- Security awareness training
- Verification of suspicious links
